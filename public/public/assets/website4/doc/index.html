Installation

To run your site, follow these instructions:

1. Unzip the downloaded package and open the "Arc/dist" folder to find all the template files.

2. You will need to upload these files to your hosting web server using FTP in order to use it on your website.

3. You should upload all or specific HTML files as per your need.

Template Structure

Below is the folder structure:

app

bootstrap - core bootstrap files

scss - core scss files

dist

css - stylesheet files

doc - documentation

fonts - font files

img - images

js - javascript files

video - video

Logotype

You can change the logo as follows:

<div class="logo-box">
    <a href="index.html"><img src="logo_path" alt=""></a>
</div>

Favicon

You can change the favicon as follows:

<link rel="icon" type="image/vnd.microsoft.icon" href="favicon_path">
<link rel="SHORTCUT ICON" type="image/x-icon" href="favicon_path">

Fonts

Arc uses 2 fonts: Open Sans & Raleway from the Google Fonts Library. You can change the fonts from /css/main.css file in below lines.

@font-face {
  font-family: "Open Sans";
  src: url('../fonts/OpenSans/OpenSans-Light.ttf') format('truetype');
  font-weight: 300;
  font-style: normal;
  font-stretch: normal;
}

Icons

You can change any icons on the site, using the classes you need, which you can take here

<i class="fa fa-cog"></i>

Images

You can change the images as follows:

<img src="/image_path" class="img-fluid" alt="">

Note- class img-fluid makes the image responsive

Header

You can change the header as follows:

<li class="menu-item">
    <div class="d-flex inside-li-wrapper align-items-center menu-link-wrap">
         <a>Pages</a>
         <i class="fa fa-angle-down"></i>
     </div>
     <div class="dropdown-menu-custom">
         <ul class="dropdown-wrap">
             <li><a href="page-about-us.html" class="main-link">About Us</a></li>
             <li><a href="page-team.html" class="main-link">Team</a></li>
             <li><a href="page-services.html" class="main-link">Services</a></li>
             <li><a href="page-contact.html" class="main-link">Contacts</a></li>
             <li><a href="page-404.html" class="main-link">404</a></li>
         </ul>
    </div>
</li>

.inside-li-wrapper > a- main menu link

.dropdown-menu-custom > .dropdown-wrap > li > main-link- main link of the dropdown menu

Background Images

You can create a parallax effect as follows:

<section class="parallax-window"
data-parallax="image_path"
data-section-type="parallax">
</section>

You can create a standard background as follows:

<section style="background-image:url('image_path')"></section>

        </div>
        <div class="item">
        </div>
        <div class="item">
        </div>
    </div>

.slick-carousel > .item- item

Portfolio

You can add, modify or delete portfolio items as follows:

<div class="grid-item development web">
</div>

.development, .web- categories

You can add, change or delete sort buttons, and also perform these actions with categories as follows:

<div class="btn-wrapper-isotope d-flex justify-content-center">
   <a class="btn-black-isotope-1 active" data-filter="*">Show All</a>
   <a class="btn-black-isotope-1" data-filter=".development">Development</a>
</div>

.btn-wrapper-isotope- button wrapper

.btn-black-isotope-1- button

Gallery

You can add, change or delete gallery items as follows:

<div class="col-lg-4 col-6 grid-item">
    <div class="img-box">
        <img src="img/portfolio/gallery-1.jpg" alt="" class="img-fluid img-gallery">
        <div class="preview-box trigger-modal
                d-flex justify-content-center align-items-center">
            <div class="search-plus d-flex justify-content-center align-items-center">
            </div>
        </div>
    </div>
</div>

.img-gallery- gallery image

Note- You can use images of different heights, because the gallery works together with the masonry grid.

Blog

You can add, change or delete blog posts as follows:

<div class="post">
</div>

.post- blog post

Note- You can use ready-made blog post templates, for example .blog-default, .blog-list or .blog-grid. Just copy the ready-made templates of blog posts and create new posts with their help..
